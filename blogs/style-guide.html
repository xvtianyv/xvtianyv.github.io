博客样式说明手册

本文档是一个动态的说明手册，展示了所有可用的文章样式及其对应的HTML源代码。

← 返回博客列表

请将此文件作为您撰写新文章的模板。下文展示了所有可用的样式模块，您可以直接复制所需的HTML代码进行使用。

正文段落

效果演示:

这是一段普通的正文。您可以在这里撰写文章的主要内容。

HTML 代码:

<p>这是一段普通的正文。您可以在这里撰写文章的主要内容。</p>

各级标题

效果演示:

这是一个二级标题

这是一个三级标题

HTML 代码:

<h2>这是一个二级标题</h2>
<h3>这是一个三级标题</h3>

内联文本样式

效果演示:

您可以让文字变为 粗体, 斜体, 或 下划线。也可以在句子中插入一小段代码，例如 const name = "Gemini";。此外，还可以为文字指定 蓝色、红色 或 绿色。

HTML 代码:

<strong>粗体</strong>
<em>斜体</em>
<u>下划线</u>
<span class="inline-code">const name = "Gemini";</span>
<span class="text-blue">蓝色</span>

代码块

效果演示:

function calculate(a, b) {
    // A multi-line code example
    return a + b;
}

HTML 代码 (注意 < > 的转义):

<pre><code>function calculate(a, b) {
    // A multi-line code example
    return a + b;
}</code></pre>

特殊内容块

效果演示:

这是一个引用块，适合引用名言或关键信息。

这是一个“分析”模块，用于记录您对问题的思考和解决方案。
